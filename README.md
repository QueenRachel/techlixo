# TechLixo

* Main branch: master
* Ruby version: 2.3.1
* Rails version: 4.2.7
* PG version: ~> 0.18

## Instalação

Para baixar o projeto digite no seu terminal:

    % git clone git@github.com:QueenRachel/techlixo.git

Agora crie o banco de dados e rode o servidor

    % bin/rake db:reset
    % bin/rails server
